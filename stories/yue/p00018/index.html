{% include audiocontrols.html index="04" prev="03" next="05" %}

{% include audiocontrols.html index="05" prev="04" next="06" %}
{% include picturecol.html index="06" %} {% include textcol_start.html %} 點解啲大人講嘅嘢咁搞笑嘅？ 我係太大個吖，定係太細個啊？ {% include textcol_end.html %}

{% include audiocontrols.html index="06" prev="05" next="07" %}

{% include audiocontrols.html index="07" prev="06" next="08" %}
{% include picturecol.html index="08" %} {% include textcol_start.html %} 我太大個，爺爺同唔到我玩騎牛牛。 我太細個，帶唔到細佬去街。 {% include textcol_end.html %}

{% include audiocontrols.html index="08" prev="07" next="09" %}
{% include picturecol.html index="09" %} {% include textcol_start.html %} 貓仔你知唔知，我而家係太大個定係太細個？ 我真係唔明白。邊個答到我？ {% include textcol_end.html %}
