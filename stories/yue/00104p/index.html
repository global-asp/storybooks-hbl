{% include picturecol.html index="02" %} {% include textcol_start.html %} 我叫小歌。 我有我自己張床，呢度係我自己嘅地方。 我鍾意喺度瞓覺。 {% include textcol_end.html %}

{% include audiocontrols.html index="02" prev="01" next="03" %}
{% include picturecol.html index="03" %} {% include textcol_start.html %} 不過我想瞓嗰陣，成日都有人撳門鐘。 個門鐘好曳曳，佢喺度「鈴鈴」、「鈴鈴」噉樣嘈住我。 {% include textcol_end.html %}

{% include audiocontrols.html index="03" prev="02" next="04" %}
{% include picturecol.html index="04" %} {% include textcol_start.html %} 我覺得好嘈。我轉一轉 身，向住第二邊。 {% include textcol_end.html %}

{% include audiocontrols.html index="04" prev="03" next="05" %}
{% include picturecol.html index="05" %} {% include textcol_start.html %} 跟住到個電話叫喇， 佢喺度「滋滋」、「滋滋」噉樣嘈住我。 {% include textcol_end.html %}

{% include audiocontrols.html index="05" prev="04" next="06" %}
{% include picturecol.html index="06" %} {% include textcol_start.html %} 之後，我又轉下個身，試下瞓覺。 {% include textcol_end.html %}

{% include audiocontrols.html index="06" prev="05" next="07" %}
{% include picturecol.html index="07" %} {% include textcol_start.html %} 真係唔開心啊！又有人喺度「咚」、「咚」噉樣嘈住我。 我瞓唔到。 {% include textcol_end.html %}

{% include audiocontrols.html index="07" prev="06" next="08" %}
{% include picturecol.html index="08" %} {% include textcol_start.html %} 「啷啷」、「啷啷」！呢次又係邊個嘈住我啊？ {% include textcol_end.html %}

{% include audiocontrols.html index="08" prev="07" next="09" %}
{% include picturecol.html index="09" %} {% include textcol_start.html %} 出面有人踩單車！佢喺度撳緊單車上面個鐘。 {% include textcol_end.html %}

{% include audiocontrols.html index="09" prev="08" next="10" %}
{% include picturecol.html index="10" %} {% include textcol_start.html %} 媽媽走過嚟，佢問我：「小歌，乜你未瞓咩？」 {% include textcol_end.html %}

{% include audiocontrols.html index="10" prev="09" next="11" %}
{% include picturecol.html index="11" %} {% include textcol_start.html %} 我望住媽媽，對眼大一大。我用對眼同媽媽講： 「我都想瞓啊！不過出面好嘈，好多聲音，我點瞓啊？」 {% include textcol_end.html %}

{% include audiocontrols.html index="11" prev="10" next="12" %}
{% include picturecol.html index="12" %} {% include textcol_start.html %} 「鈴鈴」、「滋滋」、「咚」、「啷啷」！ 我向住邊邊，都一樣畀啲聲嘈住。 {% include textcol_end.html %}
