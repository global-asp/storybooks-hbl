{% include picturecol.html index="02" %} {% include textcol_start.html %} 今日下晝，我同我個仔都着咗新衫同新褲， 一齊出去玩。 {% include textcol_end.html %}

{% include audiocontrols.html index="02" prev="01" next="03" %}
{% include picturecol.html index="03" %} {% include textcol_start.html %} 我哋搭船過海，天氣好好， 海上面有太陽，又有少少雲，天空好靚。 {% include textcol_end.html %}

{% include audiocontrols.html index="03" prev="02" next="04" %}
{% include picturecol.html index="04" %} {% include textcol_start.html %} 跟住，我哋搭巴士去沙灘。 我幫阿仔除鞋，先知道佢兩隻襪唔一樣。 {% include textcol_end.html %}

{% include audiocontrols.html index="04" prev="03" next="05" %}
{% include picturecol.html index="05" %} {% include textcol_start.html %} 我話：「唔好意思，爸爸幫你着錯襪啊。」 {% include textcol_end.html %}

{% include audiocontrols.html index="05" prev="04" next="06" %}
{% include picturecol.html index="06" %} {% include textcol_start.html %} 「係喎！」阿仔笑，拎對襪畀人睇，人人都笑。 {% include textcol_end.html %}

{% include audiocontrols.html index="06" prev="05" next="07" %}
{% include picturecol.html index="07" %} {% include textcol_start.html %} 佢真係好叻仔，喺沙上面畫咗一隻貓同兩支花， 又用沙砌咗一間屋仔。 {% include textcol_end.html %}

{% include audiocontrols.html index="07" prev="06" next="08" %}
{% include picturecol.html index="08" %} {% include textcol_start.html %} 阿仔話：「爸爸，呢間屋係我哋嘅。」 {% include textcol_end.html %}

{% include audiocontrols.html index="08" prev="07" next="09" %}
{% include picturecol.html index="09" %} {% include textcol_start.html %} 我話：「咁細間？ 爸爸、媽媽同你，三個人住一間好大嘅屋，好唔好啊？」 {% include textcol_end.html %}

{% include audiocontrols.html index="09" prev="08" next="10" %}
{% include picturecol.html index="10" %} {% include textcol_start.html %} 阿仔拍拍手。 {% include textcol_end.html %}

{% include audiocontrols.html index="10" prev="09" next="11" %}
{% include picturecol.html index="11" %} {% include textcol_start.html %} 玩到七點鐘，月亮出嚟，一樣好靚。 我哋肚餓，返屋企，日日都係噉就好喇。 {% include textcol_end.html %}

{% include audiocontrols.html index="11" prev="10" next="12" %}
{% include picturecol.html index="12" %} {% include textcol_start.html %} 我有好多嘢想同個仔做，同佢食飯，同佢去公園玩， 又有唔少嘢想幫個仔做，幫佢沖涼，唱歌畀佢聽。 {% include textcol_end.html %}

{% include audiocontrols.html index="12" prev="11" next="13" %}
{% include picturecol.html index="13" %} {% include textcol_start.html %} 轉天氣，我會問佢凍唔凍？熱唔熱？ 佢覺得凍，我會幫佢着多件衫，着多條褲。 {% include textcol_end.html %}

{% include audiocontrols.html index="13" prev="12" next="14" %}
{% include picturecol.html index="14" %} {% include textcol_start.html %} 一個好爸爸應該要明白個仔， 明白佢諗咩，知道佢鍾意咩，唔好亂估。 呢啲嘢，唔難做，但係想做得好，唔易㗎。 {% include textcol_end.html %}
